Passso para Hospedar um Projeto no GitHub Pages

  1. 1.Crie um repositório no GitHub com o mesmo nome do seu projeto
  2. 2.No terminar, inicializando o repositório local com git init.
  3. 3.Altere o nome da branch pricipal para main
  4. 4.Adicione os arquivos com git add . e faça o commit com git commit -m "first commit".
  5. 5.Conecte ao repositório remoto com:
    git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. 6.Envie os arquivos com git push
